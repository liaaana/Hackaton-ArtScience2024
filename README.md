## "Экскурсовод Ван Гог"
Веб-приложение для:

### Поиска произведений искусства по фото
Автоматизированный поиск информации (автор, год, краткое описание).

### Поиска похожих картин
Нахождение наиболее похожего произведения искусства и оценка схожести.

### Слияние картин
Возможность соединять рисунок с картиной, сохраняя её стиль.

## Запуск
Чтобы запустить приложение, выполните следующие шаги:

0. Скачайте ChromeDriver для своей версии Google Chrome 

1. Клонируйте репозиторий:
    ```
    git clone https://github.com/liaaana/ArtScience.git
    ```
2. Перейдите в директорию проекта:
    ```
    cd ArtScience
    ```
3. Установите необходимые зависимости:
    ```
    pip install -r requirements.txt
    ```
4. Запустите:
    ```
    python3 app.py
    ```

Приложение будет доступно по адресу `http://127.0.0.1:5000/`.
